# Logo & PWA Icon Files

Place these files here for full PWA and branding support:

| File                    | Size      | Usage                                |
|-------------------------|-----------|--------------------------------------|
| logo.png                | Any       | Main site logo (set in Admin > Branding > Logo URL: /images/logo/logo.png) |
| icon-192.png            | 192×192   | PWA icon (Android home screen)       |
| icon-512.png            | 512×512   | PWA splash screen icon               |
| apple-touch-icon.png    | 180×180   | iOS home screen icon                 |
| favicon.ico             | 32×32     | Browser tab favicon                  |
| favicon-16.png          | 16×16     | Small favicon                        |
| favicon-32.png          | 32×32     | Standard favicon                     |
| og-image.png            | 1200×630  | Social media share preview           |

All icons should use the Arsenal red (#EF0107) background with your club logo.
